The Legend of Zelda (NES) Remake
  • Home (current)

This game is a complete remake (exact copy) of the first dungeon from The Legend Of Zelda (NES). It also features a portal gun level.

    Controls
  • The Arrow Keys or WASD for Movement.
  • X Key corresponds to the A button on old NES controller.
  • Z Key corresponds to the B button on old NES controller.
  • Press Q to access the bonus portal gun level.
    Weapon Swap Controls
  • Key 2 = Sword on A button.
  • Key 3 = Bow on A button (When acquired in-game).
  • Key 4 = Bomb on A button (When acquired in-game).
  • Key 5 = Boomerang on A button (When acquired in-game).
